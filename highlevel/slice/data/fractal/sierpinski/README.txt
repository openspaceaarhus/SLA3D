Sierpinski Tetrahedron STL Generator

Small Java program to generate a fractal shape to a STL file. 

How to use:

Double click on sierpinski.jar in Explorer/Finder 
or type in a shell: java -jar sierpinski.jar 

command line usage: java -jar sierpinski.jar [ -depth 3 ] [ -length 1.0 ] [ sierpinski.stl ]

no parameters will bring up a dialog.

The build script is based on maven, and it should be easy to modify it to generate
other fractals based on the same method.

The output file format is the text version of STL and the files can get very huge
